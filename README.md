# Weather State 🌐

**Weather State** is a modern, responsive weather dashboard that provides real-time weather information and forecasts for any location worldwide. Built with a focus on aesthetics and user experience, it features dynamic theme switching and interactive data visualizations.

---

---


## ✨ Features

- **Real-time Weather:** Get current temperature, humidity, wind speed, and "feels like" conditions.
- **Search by City:** Quick search for weather updates in any city across the globe.
- **Auto-Location Detection:** Automatically fetch weather data for your current location using geolocation.
- **7-Day Forecast:** Detailed 7-day weather forecast to help you plan ahead.
- **Interactive Charts:** Visualized weather trends using **Chart.js**.
- **Dark Mode:** Sleek dark mode support for better night-time viewing.
- **Custom Themes:** Personalize the interface with a built-in color palette switcher.
- **Fully Responsive:** Optimized for a seamless experience on desktops, tablets, and mobile devices.

---

## 🛠️ Built With

- **HTML5 & CSS3:** For structure and premium styling.
- **JavaScript (Vanilla):** For logic and API interactions.
- **OpenWeatherMap API:** To fetch accurate, real-time global weather data.
- **Chart.js:** For interactive and dynamic data visualization.
- **BoxIcons:** For high-quality, modern icons.
